
SFX are named with a pattern that simplifies loading for our engine.
Every SFX has an ID, instead of a file name.
NOTICE: Every sound that has the intention of being used in-engine must be a .ogg file.
	Sounds not in .ogg format will crash the loading sequence.
- Linus
